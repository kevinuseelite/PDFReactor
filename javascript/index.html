RealObjects PDFreactor Web Service JavaScript API

In the subdirectories you'll find the following files:
  • lib/PDFreactor.js - The PDFreactor JavaScript client library
  • sample/simple.html - A sample demonstrating the simple integration of PDFreactor into JavaScript applications
  • sample/async.html - A sample demonstrating an asynchronous integration which is recommended for medium to large documents

The API documentation can be found in the PDFreactor installation directory at doc/webservice/javascript.html or here.

Please see the PDFreactor manual for more information.
